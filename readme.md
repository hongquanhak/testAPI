# Project Title
Resful API with NodeJS, Express and MySql
# Installing
## Clone
- Clone this repo to your local machine using `git clone https://github.com/hongquanhak/testAPI`
### Set-up
> Install npm or yarn packages
```
$ npm install
$ yarn install
```
# Running the tests
```
$ npm start
$ yarn start
```
